# Enterprise Pipeline Dashboard Specification – HubSpot

## Purpose
Defines the Enterprise Pipeline Dashboard used by Sales Leadership to oversee large, complex, multi-module opportunities.

## Audience
Sales Leadership, Exec Team, RevOps

## Principle
Enterprise visibility is achieved through reporting and grouping — not separate pipelines or parent deals.

## 1. Dashboard Overview
Create **one dashboard** titled: *Enterprise Pipeline – Consolidated View*.

Requirements:
- Include only deals flagged as Deal Type = Enterprise
- Pull data across all module pipelines
- Used for leadership review only

## 2. Core Filters
- Deal Type = Enterprise
- Deal Stage ≠ Closed Lost (unless reviewing losses)

## 3. Enterprise Pipeline Health Reports
1. Enterprise Pipeline by Company (table)
2. Enterprise Pipeline Value by Module (bar/pie)
3. Enterprise Deals by Stage (funnel)

## 4. Risk & Governance Reports
4. Without Authority
5. At Risk or Stalled
6. Undefined Decision Timeline

## 5. Commercial Exposure Reports
7. Deals with Discounts
8. Discounted ACV summary

## 6. Optional Reports
9. Enterprise Opportunity Groups
10. Win/Loss Analysis

## 7. Must Not Include
- Calls, emails, meetings, SDR activity metrics

## 8. Review Cadence
- Weekly: leadership review
- Monthly: risk assessment
- Quarterly: win/loss analysis
