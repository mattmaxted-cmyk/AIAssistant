# HubSpot Sales Build – Acceptance Checklist

## Purpose
Verify HubSpot configuration matches approved Sales Playbook and build specification before go-live.

## 1. Deal Pipeline Structure
- [ ] Single pipeline exists
- [ ] Stages match approved list
- [ ] No extra/renamed stages
- [ ] Correct stage order

## 2. Deal Properties
- [ ] Qualification
- [ ] Demo
- [ ] Pricing & Decision
- [ ] Discount governance
- [ ] Handover & onboarding

## 3. Stage Gating
- [ ] Demo Booked gated by Procurement Problem Confirmed
- [ ] Pricing Presented gated by Demo Delivered + Authority Present
- [ ] Closed Won gated by Contract Signed + Handover Completed

## 4. Workflows
Inbound, Qualification, Demo, Pricing, Stalled, Discount, Closed Won flows verified.

## 5. Email Templates
All required templates loaded and match wording.

## 6. Dashboards
Pipeline, governance, activity dashboards configured.

## 7. Permissions
Managers approve discounts only.

## 8. Testing
All paths tested end-to-end.
