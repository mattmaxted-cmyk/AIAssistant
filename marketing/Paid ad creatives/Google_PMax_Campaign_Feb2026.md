### UTM Parameter Breakdown

| Parameter | Value | Notes |
|-----------|-------|-------|
| utm_source | `google_ads` | Traffic source |
| utm_medium | `cpc` | Cost-per-click |
| utm_campaign | `pmax_procurement` | Campaign identifier |
| utm_content | `asset_group_core` | Asset group variant |

**Note:** Google Ads auto-tags with `gclid` for conversion tracking. The UTMs above are for manual/backup tracking if needed. Ensure auto-tagging is enabled in Google Ads settings.

## Step 8: Conversion Tracking Setup

### Primary Conversion Action

| Setting | Value |
|---------|-------|
| Conversion name | `Demo Form Submission` |
| Category | Lead |
| Value | £100 (estimated lead value) |
| Count | One (per click) |
| Click-through window | 30 days |
| View-through window | 1 day |

### Implementation Options

**Option A: Google Tag Manager (Recommended)**
1. Create a trigger for form submission on `/lp/procurement`
2. Fire Google Ads conversion tag on successful submission
3. Test with GTM Preview mode

**Option B: Google Ads Tag Direct**
1. Place global site tag on all pages
2. Place event snippet on thank-you/confirmation page

### Enhanced Conversions

Enable Enhanced Conversions for better attribution:
1. Go to Tools & Settings > Conversions
2. Select your conversion action
3. Enable Enhanced Conversions
4. Choose "Google Tag Manager" or "Global site tag"
5. Pass email/phone from form submission

## Step 12: Asset Performance Ratings

After 1-2 weeks, Google will rate your assets:

| Rating | Meaning | Action |
|--------|---------|--------|
| Best | Top performer | Keep, create variations |
| Good | Performing well | Keep |
| Low | Underperforming | Replace with new variation |
| Pending | Not enough data | Wait for more impressions |

**Goal:** Aim for 0 "Low" rated assets after week 2.
