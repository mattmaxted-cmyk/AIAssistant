### 5. A/B Test: Advantage+ Creative

**Objective:** Test whether Advantage+ creative improves or hurts performance for native UI format ads.

**Hypothesis:** Native formats (Notes, WhatsApp, iMessage) may perform worse with Advantage+ ON because Meta's auto-enhancements could break the authentic look.

#### Test Setup

**Test creative:** Choose 1 creative to test (recommend a Notes App format)

| Version | Advantage+ Creative | Notes |
|---------|---------------------|-------|
| A (Control) | OFF | Creative runs exactly as designed |
| B (Test) | ON | Meta may auto-enhance |

**How to set up:**
1. Create the ad with Advantage+ OFF
2. Duplicate the same ad
3. Turn Advantage+ ON for the duplicate
4. Both run in the same ad set, Meta distributes spend

**Success metrics:**
- CTR (Click-through rate)
- CPC (Cost per click)
- Conversion rate (demo bookings)
- Spend distribution (is Meta favouring one?)

**Test duration:** Minimum 7 days or until each version has 1,000+ impressions

**Decision framework:**
- If Advantage+ ON wins → Consider enabling on all myth-bust format ads
- If Advantage+ OFF wins → Keep all native format ads with Advantage+ OFF
- If no significant difference → Default to OFF for creative control
