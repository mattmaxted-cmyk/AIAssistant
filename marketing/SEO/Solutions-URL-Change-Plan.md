# Solutions Page URL Change Plan

**Date:** 20 March 2026  
**Owner:** Michael (dev), Matt (content + GSC)  
**Objective:** Add construction-specific keywords to all Solutions page URLs to improve organic search relevance, replicating the pattern already working on `/solutions/construction-hr`.

---

## URL Changes

| Module | Current URL | New URL | 301 Redirect Required |
|---|---|---|---|
| HR | `/solutions/construction-hr` | No change | No |
| Procurement | `/solutions/procurement` | `/solutions/construction-procurement-software` | Yes |
| Payments (Certify) | `/solutions/payments` | `/solutions/construction-payment-software` | Yes |
| Health & Safety | `/solutions/health-and-safety` | `/solutions/construction-health-safety-software` | Yes |
| Site Management | `/solutions/site-management` | `/solutions/construction-site-management-software` | Yes |

---

## For Each Changed Page

1. Set up 301 redirect from old URL to new URL **before or at the same time** as the URL change — never after
2. Update the page H1 to include the target keyword (e.g. "Construction Procurement Software")
3. Update the meta title (e.g. "Construction Procurement Software | StoneRise")
4. Update all internal links (nav, blog posts, CTAs) to point directly to the new URL — don't rely on the redirect for internal links
5. Update XML sitemap
6. Submit new URLs in Google Search Console once live

---

## Target Keywords per Page

| Page | H1 Target | Meta Title Target |
|---|---|---|
| `/solutions/construction-procurement-software` | Construction Procurement Software | Construction Procurement Software \| StoneRise |
| `/solutions/construction-payment-software` | Construction Payment Software | Construction Payment Software \| StoneRise |
| `/solutions/construction-health-safety-software` | Construction Health & Safety Software | Construction Health & Safety Software \| StoneRise |
| `/solutions/construction-site-management-software` | Construction Site Management Software | Construction Site Management Software \| StoneRise |

---

## Notes

- HR page is already ranking — do not touch the URL, H1, or meta title
- 301 redirects pass existing link equity to the new URL — Google will update its index within a few weeks of the change
- After redirects are live, monitor Google Search Console for crawl errors on the old URLs
