### 2. Problem Block — 3 Pain Points
Present as short, punchy statements. Each should feel like something the reader recognises from their own day:

- Sending quote requests by email means responses come back in different formats, with different inclusions and exclusions. Comparing them accurately takes longer than it should and the risk of picking the wrong one is high
- When quotes are managed in email threads, there is no record of which supplier quoted what and when. If a dispute arises or an auditor asks, the information is almost impossible to recover
- Without a structured process, the same materials get bought from the same supplier out of habit rather than price. Over a year, across hundreds of orders, that adds up

### 3. Solution Block — How StoneRise Procurement Works
Four features, each as a heading plus one sentence of benefit:

- Multi-supplier RFQ — send a single quote request to multiple suppliers simultaneously, with a consistent format that makes responses directly comparable
- Intelligent quote split — StoneRise allows quote requests to be split across suppliers at line item level, so the cheapest price for each material is selected rather than accepting one supplier for the whole order
- Quote comparison view — all supplier responses appear in a side-by-side view so procurement teams can identify the best price for each line without building a spreadsheet
- Quotation extraction AI — StoneRise reads supplier quote documents and extracts line items automatically, removing the manual data entry that slows the comparison process down
